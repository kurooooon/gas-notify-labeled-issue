## Description

GAS tool for
- Notify GitHub issue which is labeled specific label to Slack channel.

## SETUP

### Installation

```
$ yarn install
$ yarn clasp create {project name} --rootDir ./src
```

### Push the scripts to GAS

```
$ yarn clasp push
```
